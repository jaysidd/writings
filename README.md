## How this is organized

Each article lives in its own folder at the repo root:

```
writings/
├── agent-lab/
│   └── index.md
└── README.md
```

GitHub Pages serves `index.md` as the article page, giving every piece a clean URL of the form `https://jaysidd.github.io/writings/<slug>/`. That URL is what gets pasted into Medium's import tool, which auto-sets it as the canonical link.
